## How to Take Screenshots

### For React App:
1. Run `npm start` in student-registration-react folder
2. Open http://localhost:3000
3. Take screenshot of the form
4. Fill with invalid data and take validation screenshot
5. Submit successfully and take success page screenshot

### For API:
1. Run `npm start` in library-backend folder
2. Open Postman
3. Take screenshots of different API requests
4. Take screenshot of server console output
